逻辑导出与导入

普通方式exp/imp
exp
1）导出全库，将数据库orcl完全导出（需要DBA权限）
exp \'sys/Oracle_1@orcl as sysdba\' file=/oracle/11g/export/full.dmp full=y

2）按用户导出，将数据库orcl中scott用户下的所有对象导出，如果没有DBA权限，无法导出其他用户的对象，仅能导出自身的
exp scott/scott@orcl file=/oracle/11g/export/scott.dmp owner=\(scott\)

3）按表导出， 将数据库orcl中scott用户的emp、emp1表导出
exp scott/scott@orcl file=/oracle/11g/export/emp1.dmp tables=\(emp,emp1\)

exp system/Oracle_1@orcl file=/oracle/11g/export/emp1.dmp tables=scott.emp1

4）按表导出，可以加上where字句
exp scott/scott@orcl file=/oracle/11g/export/emp1.dmp tables=\(emp1\) query=\"where empno='8888'\"

5）按表空间导出，将数据库orcl中的tbs01表空间导出（需要DBA权限）
exp scott/scott@orcl file=/oracle/11g/export/tbs.dmp tablespace=\(tbs01\)

imp导入，！！！！注意导入时连接的数据库，千万别直接粘贴复制，忘记修改连接的数据库就麻烦了！！！！
imp scott/scott@orcl file=/oracle/11g/export/emp1.dmp tables=emp1
imp scott/scott@orcl file=/oracle/11g/export/emp1.dmp tables=emp1 ignore=y  ##如果记录已存在，忽略，还是新增导入，所以可能会有重复的数据

可以从按照用户导出的文件中导入某张表，如
imp scott/scott@orcl file=/oracle/11g/export/scott.dmp tables=emp1

可以将一个用户拥有的数据导入到另一份用户
imp system/Oracle_1@orcl file=/oracle/11g/export/emp1.dmp fromuser=scott touser=scott1;



数据泵方式expdp/impdp
SQL> create directory dmp_dir as '/oracle/11g/export';

Directory created.

SQL> grant read,write on directory dmp_dir to scott;

Grant succeeded.

1）全库导出
expdp system/Oracle_1@orcl1 directory=dmp_dir dumpfile=fulldb.dmp full=y

2）按用户导出
expdp system/Oracle_1@orcl1 directory=dmp_dir dumpfile=scott.dmp schemas=scott,scott1

3）按表导出
expdp system/Oracle_1@orcl1 directory=dmp_dir dumpfile=tables.dmp tables=scott.emp,scott.dept

4）按表导出，加上where字句
expdp system/Oracle_1@orcl1 directory=dmp_dir dumpfile=tables.dmp tables=scott.emp query=\"where empno='8888'\"

5）按表空间导出
expdp system/Oracle_1@orcl1 directory=dmp_dir dumpfile=tablespace.dmp tablespaces=rman


导入，！！！！注意导入时连接的数据库，千万别直接粘贴复制，忘记修改连接的数据库就麻烦了！！！！
1）按全库导入
impdp system/Oracle_1@orcl directory=dmp_dir dumpfile=fulldb.dmp full=y

2）按用户导入
impdp system/Oracle_1@orcl directory=dmp_dir dumpfile=scott.dmp schemas=scott

导入到不同的用户
impdp system/Oracle_1@orcl directory=dmp_dir dumpfile=scott.dmp remap_schema=scott:scott1 table_exists_action=replace

table_exists_action=replace表示替换已存在的数据
table_exists_action=append表示追加数据

3）按表导入
impdp system/Oracle_1@orcl directory=dmp_dir dumpfile=tables.dmp tables=scott.emp

4）按表空间导入
impdp system/Oracle_1@orcl directory=dmp_dir dumpfile=tablespace.dmp tablespaces=rman

导入到不同的表空间
impdp system/Oracle_1@orcl directory=dmp_dir dumpfile=tablespace.dmp remap_tablespace=rman:rman1
