创建测试表
1，学生表
Create table Courses  -- 成绩表
( SID  varchar2(5),  --学号
  Clase varchar2(8),  --课程
  Score int  -- 分数
);
2，成绩表
Create table Student  --学生表
( SID  varchar2(5),  --学号
   NAME varchar2(8),  -- 姓名
  SEX  varchar2(2)  --性别
);
   
1,内连接
内连接查询返回满足条件的所有记录，默认情况下没有指定任何连接则为内连接
第一种写法
Select c.sid,s.name,c.clase,c.score
From Student s
Inner join
Courses c
on s.sid=c.sid
第二种写法
Select c.sid,s.name,c.clase,c.score
From Courses c
Inner join
Student s
on c.sid=t.sid
第三种写法
Select c.sid,s.name,c.clase,c.score
From Courses c, Student s
Where c.sid=s.sid

2,左外连接
左外连接查询不仅返回满足条件的所有记录，而且还会返回不满足连接条件的连接操作符左边表的其他行
左面是学生表
Select c.sid,s.name,c.clase,c.score 
From Student s 
left join 
Courses c 
on s.sid=c.sid
查询结果如下图：强加成绩表空白，同时返回了缺成绩的学生姓名
  强加成绩表空白，同时返回了缺成绩的学生姓名
左面是成绩表
Select c.sid,s.name,c.clase,c.score
From Courses c
left join
Student s
on c.sid=s.sid
查询结果如下图：强加学生表空白，同时返回了缺姓名的成绩

3,右外连接
右外连接查询不仅返回满足条件的所有记录，而且还会返回不满足连接条件的连接操作符右边表的其他行
左面是学生表
Select c.sid,s.name,c.clase,c.score 
From Student s 
right join 
Courses c 
on s.sid=c.sid
